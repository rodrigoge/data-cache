# Spring Data Cache
A study project to use Redis in API with Spring

### 🎯 Objectives
The objective of this project was build a Spring API using Redis cache to load various data.

### 🛠️ Technologies
- Java 22
- Spring Boot 3
- PostgreSQL
- Docker
- Redis

### 📝 API Documentation 

#### User authentication

#### Get all products
```http
  GET /products
```

#### Get all products with cache
```http
  GET /products/cache
```
